# SuperKart Sales Prediction

## Project Overview
This project builds and deploys a machine learning model to predict product sales using store and product attributes.

## Features
- Data preprocessing
- Model training (Gradient Boosting)
- Hyperparameter tuning
- Deployment using Streamlit
- CI/CD using GitHub Actions
- Hosting on Hugging Face Spaces

## Tech Stack
- Python
- Scikit-learn
- Streamlit
- Docker
- GitHub Actions
- Hugging Face

## Links
- GitHub: https://github.com/VishalSapkalAutomationStack/Superkart
- Hugging Face App: https://huggingface.co/spaces/Vishal1986/Superkart

## Automation
The pipeline automatically:
- Trains model
- Saves model
- Deploys to Hugging Face

Triggered on every push to main branch.
